Código-Claude-Livre-Ruflo

Experimental Python-based orchestration framework designed to create a hybrid multi-agent AI ecosystem using free cloud-based LLM services.

---

Overview

This project aims to automate and coordinate multiple AI models through an agentic orchestration architecture using:

- Free Claude Code
- Rufflo
- Python automation
- Free cloud AI providers
- Hybrid multi-agent workflows

The objective is to reduce local hardware dependency by leveraging free cloud inference services while maintaining local orchestration and automation.

---

Main Goals

- Automate AI agent workflows
- Create a lightweight orchestration system
- Reduce hardware limitations
- Enable multi-model AI coordination
- Experiment with free cloud-based LLM infrastructures
- Develop fallback and routing strategies between AI models

---

Planned Architecture

Local Machine (Python Orchestrator)
↓
Free-Claude-Code Compatibility Layer
↓
Cloud-based Free LLMs
(DeepSeek / GLM / Kimi / Gemini / Qwen)
